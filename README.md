<h1 style="text-align: center">Python API for PhoneDB.net</h1>

<h3 style="text-align: center">**phonedb_api** 是一个简单而优雅的 Python 库, 实现了访问 PhoneDB.net 的 API.</h3>

# 功能
## API
1. [x] 获取信息页
2. [x] 获取最新信息ID
3. [ ] 搜索信息

## 特性
1. [x] 支持异步调用
2. [x] 支持批量调用
3. [x] 支持本地数据库缓存
